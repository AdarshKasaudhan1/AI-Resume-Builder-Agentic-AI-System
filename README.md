🤖 AI Resume Builder – Agentic AI System

An intelligent 4-Agent AI Resume Builder that tailors resumes to specific job descriptions using LangGraph, LangChain, Groq Llama 3.3, and Streamlit. The system evaluates resumes, identifies missing skills, recommends improvements, and generates ATS-friendly resumes automatically.

✨ Features
📄 Resume Parsing (PDF/DOCX)
🔍 Job Description Analysis
⭐ AI Resume Match Scoring
💡 Skill Gap Analysis
🚀 Project Recommendations
📝 ATS-Friendly Resume Generation
🌐 Streamlit Web Interface
🤖 Agent Workflow
Agent 1: Job Description Parser
Agent 2: Resume Evaluator
Agent 3: Gap Analyst & Project Recommender
Agent 4: Resume Writer
